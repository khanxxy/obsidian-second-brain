# Projects

All active, paused, and finished projects.

Use `Projects/.template/` when starting a new project.

---

## How To Start A Project

1. Copy `Projects/.template/` to `Projects/<Project Name>/`.
2. Fill in `Context.md`.
3. Add the project to [[NOW]].
4. Start a first entry in `HANDOFF.md`.
