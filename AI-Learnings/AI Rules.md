# AI Rules & Learnings

> Rules for all AI agents. Learned from mistakes — this is what works.
> Tool-agnostic: Applies to Claude, Codex, Cursor, and whatever comes next.

---

## 1. Read VAULT.md First

Every task starts by reading `VAULT.md`. Navigate from there. Don't guess the vault structure — follow the links.

**Why:** Agents that skip this waste context on irrelevant files or create duplicates.

## 2. Write Back What You Learn

If you produced knowledge, created a decision, or discovered something — write it into the vault before the session ends.

**Where:**
- Knowledge → `Knowledge/`
- Decision → `Decisions/`
- Person → `People/`
- Learning → this file
- Everything → `Changelog.md`

## 3. Link, Don't Duplicate

Before creating a new file, check if one already exists on the topic. If it does, add to it. Use `[[wiki-links]]` to connect related notes.

**Why:** Duplicate notes create confusion. Linked notes create a knowledge graph.

## 4. Use the Templates

New project? Copy `Projects/.template/`. Don't invent a new structure every time.

## 5. Changelog Is Mandatory

Every session that changes the vault must add an entry to `Changelog.md`. No exceptions.

## 6. Don't Over-Organise

A note in `Inbox/` is better than no note at all. Sorting happens later.
