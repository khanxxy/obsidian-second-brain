# AI Rules

> Stable rules for working with AI agents in this vault.
> See also: [[AGENTS]] for session-level protocol.

## Core Rules

1. **Read before writing.** Always read `NOW.md` and the relevant `HANDOFF.md` before starting work.
2. **Update before leaving.** Always update `NOW.md`, `HANDOFF.md`, and `Work Log/` before ending a session.
3. **Don't guess state.** If you don't know the current status of something, read the file — don't invent an answer.
4. **Surgical edits only.** Change only what you're asked to change. Don't reorganize, rename, or "improve" things that weren't part of the request.
5. **No silent failures.** If something goes wrong, say so explicitly. Don't report success without verifying it.

## Memory Between Sessions

AI agents have no memory between sessions by default.  
This vault compensates for that:

- `NOW.md` = current project state
- `HANDOFF.md` = what the last agent did and what you need to do next
- `Work Log/` = full session history

Use them. Update them. That's how continuity works.

## When in Doubt

Ask one clarifying question. Don't guess. Don't do extra work that wasn't requested.
