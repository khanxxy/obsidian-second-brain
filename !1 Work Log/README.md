# Work Log

Daily log of human and agent sessions.

One file per day:

```text
YYYY-MM-DD.md
```

Use [[!1 Work Log/template]] when creating a new day.

This is separate from [[!2 Briefings/Index]], which is for bot outputs and reports.

---

## What Belongs Here

- Interactive agent sessions
- Commits and operational changes
- Decisions made during work
- Verification results
- Open follow-up tasks from today's work

## What Does Not Belong Here

- Automated news feeds
- Bot briefings
- Raw research dumps
- Long-term knowledge pages
