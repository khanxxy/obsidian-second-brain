# Contributing

This is a template vault — contributions welcome.

## How to contribute

1. Fork the repo
2. Create a branch: `feat/your-improvement`
3. Make your changes
4. Open a PR with a clear description

## What we're looking for

- Better templates for projects, decisions, or people
- Improved agent rules in `AGENTS.md`
- Cleaner navigation structure
- Real-world usage patterns that make the vault more useful

## What we're NOT looking for

- Private data or example businesses
- Plugin-specific features — keep it plugin-agnostic
- Over-engineering — this vault should stay minimal

## Style guide

- All files in English
- Minimal structure — only add what's actually needed
- Wiki-links (`[[like this]]`) for internal navigation
